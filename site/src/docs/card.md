## Card Props

| 名称         | 类型     | 默认值 | 说明               |
| ------------ | -------- | ------ | ------------------ |
| title        | `string` | 无     | 卡片标题           |
| headerBg     | `string` | 无     | 卡片头部的背景图片 |
| headerHeight | `string` | 无     | 卡片头部的高度     |
