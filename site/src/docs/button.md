## Button Props

| 名称    | 类型            | 默认值  | 说明               |
| ------- | --------------- | ------- | ------------------ |
| primary | `true \| false` | `false` | 按钮是否为主要按钮 |
| success | `true \| false` | `false` | 按钮是否为成功按钮 |
| info    | `true \| false` | `false` | 按钮是否为信息按钮 |
| warning | `true \| false` | `false` | 按钮是否为提示按钮 |
| danger  | `true \| false` | `false` | 按钮是否为警告按钮 |
| plain   | `true \| false` | `false` | 按钮是否为简约风格 |
